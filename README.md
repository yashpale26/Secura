Secura - Comprehensive Mobile Security Application
Secura is an all-in-one mobile security platform designed to protect Android users from digital threats while enhancing privacy. It integrates multiple protective modules—from malware scanning to secure data storage—into a single, user-friendly interface to eliminate the need for multiple third-party apps.

🚀 Key Features

🛡️ Core Security
  Malware Scanner: Scans installed applications and files for malicious activities.
  Network Scanner: Analyzes Wi-Fi networks for vulnerabilities or suspicious activities.
  Secure Browser: Blocks suspicious links and unsafe websites for a safer browsing experience.
  Call & SMS Scanner: Filters spam and potentially harmful messages to prevent fraud.
  
🔐 Privacy & Protection
  App Lock & Gallery Vault: Secures private media and specific applications with restricted access.
  Intruder Selfie: Automatically captures an image of anyone attempting unauthorized access to the device.
  Secure Notepad: Encrypted storage for sensitive notes.

📊 Device Monitoring
  Security Monitor: Real-time status updates on the overall security of the device.
  Battery Monitor: Tracks battery usage and health in real time.
  App Management: Provides a centralized view for managing installed applications.
  
🤖 Interactive Assistance
  Cybersecurity Chatbot: Explains complex cybersecurity terms to the user.
  Assistant Chatbot: Provides guidance on how to use the app's features effectively.

🛠️ Technical Stack
  Platform: Android
  Language: Java
  IDE: Android Studio
  Database: SQLite (local storage for encrypted notes, vault, and profiles)
  UI Framework: Material Design Components

📋 Requirements
  Development Environment
  Operating System: Windows, macOS, or Linux.
  RAM: Minimum 4GB (8GB recommended for Android Studio).
  Storage: 10GB free space for SDKs and project files.
  Target Device
  Android Smartphone: For testing and running the application.

🏗️ System Architecture
  The project followed the Spiral Model of development, ensuring continuous enhancement through multiple iterations.
  Iteration 1: Development of Malware and Network Scanners.
  Iteration 2: Integration of App Lock, Secure Notepad, and Gallery Vault.
  Iteration 3: Addition of Secure Browser, Security Monitor, and Chatbots.
